Axialis Universal Pro - HARDWARE & NETWORK SET
==============================================
Version 1.0 - August 27, 2021

PRESENTATION
------------
This icon set is made for use in application ribbons, toolbars & menus. The style is sharp, modern & dynamic, suitable for applications on all platforms such as Windows, Mac OSX, Linux, iOS, Android.

Using Axialis IconGenerator, create all kinds of vector and raster icons in minutes. Supported vector formats are SVG, XAML, PDF, EPS, PS, Icon Fonts. You can also generate raster images at all standard and custom sizes in PNG, BMP, JPEG. Windows ICO and Mac ICNS formats are also supported.

The following vector formats are included in the set: AI, SVG, XAML, EPS, PS, Icon Fonts (EOT, WOFF, SVG, TTF, OTF, CSS). Bitmap PNG files are also provided in 24x24, 48x48, 72x72 pixels for the large versions and 16x16, 32x32 and 48x48 for the small versions. 

CONTENT OF THE SET
------------------
This set covers all your needs to illustrate WORD PROCESSING functions and objects such as computer, server, network, cpu, firewall, phone, connection, screen, tablet, usb-key and more. 

273 unique icons are available in the set. To let you cover all your needs, overlays images (102 files) are provided in IconGenerator. This lets you create all the derivative you need.

This ZIP file contains all the icons in several formats organized in folders as follows:

- Folder "AI" - Contains the icons as individual Adobe Illustrator files (2 x 273 files).
- Folder "AI (all in one)" - Contains all the icons in a single Adobe Illustrator file (2 files).
- Folder "EPS" - Contains the icons as individual EPS files (2 x 273 files).
- Folder "FONT" - Contains the icons as Font files. Contains also CSS and HTML test files (2 x 9 files).
- Folder "PS" - Contains the icons as individual PS files (2 x 273 files).
- Folder "SVG" - Contains the icons as individual SVG files (2 x 273 files).
- Folder "XAML" - Contains the icons as individual XAML files (2 x 273 files).
- Folder "PNG" - Contains 3 subfolders with PNG icons at size 1x, 2x and 3x (6 x 273 files).

If not already done, we strongly recommend installing Axialis IconGenerator application. Once installed you can generate derivative icons, customize them and create many different file formats.

To activate the icon set in IconGenerator follow the procedure below:

   - Launch IconGenerator.
   - Be sure to have a working internet access.
   - Select the Icon Set in the list on left.
   - Once the icons are displayed, click the red button "LICENSE" and choose "Activate License (enter key)".
   - Enter your key, specify your registration information if needed and click "Activate License". Done! :)
   - Now you can generate all your icon files without limitation.
