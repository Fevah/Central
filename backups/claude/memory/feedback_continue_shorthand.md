name: "c" means continue
description: When user sends just "c" or "continue", it means keep going with the current task — don't ask what to do
type: feedback
---

When the user sends just "c" it means "continue" — keep going with whatever the current task is. Don't stop to ask what to do next.

**Why:** User works fast and doesn't want to be interrupted with confirmation prompts.

**How to apply:** Treat "c" the same as "continue". Pick up the next logical step and keep building.
