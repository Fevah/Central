PicOS Config Files

Upload one or more .txt PicOS config files (e.g. MEP-91-CORE02.txt). The parser will extract all interfaces, VLANs, BGP, VRRP, routing, and QoS config.

Switch Guide (Excel)

Upload switch_guide.xlsx to import switch inventory and port-level connections. Column names are auto-detected — works with any reasonable header naming.

Command-line Import

You can also run imports directly from the command line:

cd /central

# Load all .txt configs in current directory + guide
python parser/db_loader.py

# Configs only
python parser/db_loader.py --configs-only --config-dir .

# Guide only
python parser/db_loader.py --guide-only --guide switch_guide.xlsx
